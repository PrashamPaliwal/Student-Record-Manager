# Student-Record-Manager

A beginner-level Python project to practice **file handling** and **CRUD operations**.  
This program manages student records (Name, DOB, Roll Number) using a text file as storage.  
It demonstrates how to build a menu-driven system for adding, viewing, searching, and deleting data.

---

## 📂 Repository Contents
- **student_data.py** → Final polished program (run this file).  
- **student_data(RAW).py** → Original draft version, kept intentionally to show the development journey.  
- **README.md** → Documentation for the project.  

⚠️ Note: The text file used for storage (`student data`) is **not uploaded**. It will be **created automatically** when the program runs.

---

## 🚀 Features
- Add new student records (Name, DOB, Roll Number).  
- View all student records.  
- Search for a student by roll number.  
- Delete a student record by roll number.  
- Menu-driven interface for easy use.
